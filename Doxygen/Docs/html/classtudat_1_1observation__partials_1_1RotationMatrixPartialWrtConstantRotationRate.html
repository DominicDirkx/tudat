Detailed Description

Class to calculate a rotation matrix from a body-fixed to inertial frame w.r.t. a constant rotation rate.

Class to calculate a rotation matrix from a body-fixed to inertial frame, as computed by the SimpleRotationalEphemeris class, w.r.t. the constant rotation rate.
