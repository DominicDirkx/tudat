Detailed Description

Class that computes the orientation of a body as a function of the current state and time.

Class that computes the orientation of a body as a function of the current state and time. The functionality of this class is distinct from the RotationalEphemeris, in the sense that it is only defined during the numerical propagation, as it relies on the current state of the environment for the computation of the rotations.
