Detailed Description

template<typename StateScalarType = double, typename TimeType = double>
class tudat::ephemerides::TabulatedCartesianEphemeris< StateScalarType, TimeType >

Class that determines an ephemeris from tabulated data.

Class that determines an ephemeris from tabulated data, by using numerical interpolation of this data. Required input to this class is a OneDimensionalInterpolator, which may be reset. This class may for instance be used for setting the numerically integrated state of a body as its 'new' ephemeris
