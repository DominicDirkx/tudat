Detailed Description

template<typename DataType>
class tudat::unit_tests::TestBody< 1, DataType >

Test body class.

This class is a template specialization of the general TestBody class for scalar positions/velocities (i.e. doubles, floats etc. instead if Eigen-types).

Template Parameters
DataTypeThe data type of all of the internally stored state-related vectors and time.
