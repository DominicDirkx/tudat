Detailed Description

Class for providing settings for aerodynamic coefficient model of control surfaces.

Class for providing settings for automatic control surfaces aerodynamic coefficient model creation. This class is a functional (base) class for settings of aerodynamic coefficient models that require no information in addition to their type. Aerodynamic coefficient model classes defining requiring additional information must be created using an object derived from this class. The aerodynamic reference quantities of this interface are not stored here, but are instead implicitly assumed to be equal to the AerodynamicCoefficientSettings hosting it.
