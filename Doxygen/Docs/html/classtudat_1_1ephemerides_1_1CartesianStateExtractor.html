Detailed Description

Cartesian State extractor class.

This class contains the functionality of extracting the Cartesian elements from a parsed data line map.

Extract the Cartesian elements.

Returns a CartesianElements object containing the cartesian elements found in the input data line map. If one of the elements is not present, the function throws an exception.
