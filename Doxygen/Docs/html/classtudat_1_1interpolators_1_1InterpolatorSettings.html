Detailed Description

Base class for providing settings for creating an interpolator.

Base class for providing settings for creating an interpolator using the createInterpolator function. This base class is functional, i.e. may be used directly for interpolator types requiring no additional information. For interpolators that do require more information, a derived class is provided.
