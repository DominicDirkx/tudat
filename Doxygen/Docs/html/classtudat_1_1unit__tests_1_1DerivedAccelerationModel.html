Detailed Description

template<typename AccelerationDataType = Eigen::Vector3d, typename PositionDataType = Eigen::Vector3d, typename TimeDataType = double>
class tudat::unit_tests::DerivedAccelerationModel< AccelerationDataType, PositionDataType, TimeDataType >

Acceleration model derived class.

This class serves as an example of a derived class from the AccelerationModel base class. This class contains an implementation that computes acceleration based on a given position and time. The context of this class is somewhat specific to make it more understandable, though derived classes of the AccelerationModel base class can be completely general. It should be noted that this class should NOT be used as is in the rest of the library, and is only used in conjunction with unit tests.

Template Parameters
AccelerationDataTypeData type used to represent accelerations (default=Eigen::Vector3d).
PositionDataTypeData type used to represent positions (default=Eigen::Vector3d).
TimeDataTypeData type used to represent time (default=double).
