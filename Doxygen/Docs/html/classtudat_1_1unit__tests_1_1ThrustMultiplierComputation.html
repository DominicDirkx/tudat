Detailed Description

Class that is used in the following unit test to check whether the parameterized thrust guidance is working correctly. This class sets (using an algorithm with no physical basis). It computes a 'regular' guidance input, as well as a thrust multiplier (throttle) if required. The regular guidance input is used as the Mach number when interpolating the thrust tables.
