Detailed Description

Interface class for estimation of spherical harmonic gravity field cosine coefficients.

Interface class for estimation of spherical harmonic gravity field cosine coefficients. Interfaces the estimation with the member coefficients of an instance of the SphericalHarmonicsGravityField (or nominal coefficients of TimeDependentSphericalHarmonicsGravityField).
